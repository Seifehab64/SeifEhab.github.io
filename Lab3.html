Lab 3 - Responsive Design

This is a simple example of a responsive web page using Bootstrap.

This is our first responsive website. In this lab, we explore the principles of responsive web design using Bootstrap. Responsive design ensures that web pages look good on all devices, from desktops to tablets to mobile phones. By using Bootstrap's grid system and responsive utilities, we can create flexible layouts that adapt to different screen sizes and orientations.
